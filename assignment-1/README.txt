The assisgment was to perform data cleaning on Toyota_corolla dataset in R.
The steps to be performed were:

1) Check for missing values, handle them accordingly. Imputation and removing methods were used
2) Change the class type of wrongly defined columns
3) Create dummy variables for values which may be required in regression models or any analysis
4) Check if there are any outliers in the dataset and remove if they seem fit after understanding the structure and summary of dataset
